# sztu-toolkit

Kaede, 一个大一新生自制的 SZTU 小工具, 欢迎查看源代码或者自行进行修改!

## 技术栈

Vue3, Vite, Electron

## 程序配置文件

在根目录的`configs`文件夹中, 自行修改即可

## Project Setup

### Install

```bash
$ yarn
```

### 开发

```bash
$ yarn dev
```

### 构建

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
